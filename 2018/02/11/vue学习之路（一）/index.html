vue学习之路（一）利用vue-cli构建vue项目

  • vue

利用vue-cli来开发vue项目第一步

vue是什么：（官方解释）Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

其实就是一个前端框架。

vue-cli是什么：vue-cli是Vue的脚手架工具,打个比方，就好比就要修建房子，首先要做的就是将房子的整体结构先修好，然后在往里面填东西。vue-cli就提供了这样一个功能，将webpack等打包程序集为一体，让我们开发vue项目更加方便快捷。

本教程以window版本安装为例，其他系统大同小异。

一.
要使用vue-cli，首先得安装node.js,我相信学习前端的小伙伴多半电脑上都安装了node.js，这里我们也只使用node.js的npm包管理功能。若没有安装，则到node.js官网下载安装即可，安装步骤和安装QQ没什么区别。检查安装成功与否：win+R键输入cmd弹出命令行窗口，在命令行窗口输入node -v，若显示node的版本号则安装成功。因为现在npm管理工具已经和node.js集成了，所用不用再单独下载了。

二.
接下来就是使用vue-cli了。
要使用Vue-cli，首先就得安装vue-cli。就在刚刚的那个命令行窗口输入npm install -g vue-cli，-g的意思是全局安装。由于网络限制原因，安装过程可能有些缓慢，在意的同学可百度npm淘宝镜像不在意的同学不用管。

若如图显示，则恭喜安装成功。

三.
接下来在电脑的任意位置建立一个文件夹，这里我取名为test。打开test文件夹：

然后在当前文件夹任意位置按住shift键然后鼠标右击，会出现在此处打开命令行工具的选项：

点击打开命令行工具，输入vue init webpack对该项目以webpack的打包模板进行初始化。网络原因，这个过程可能要等一会儿。

输入Y，继续执行

项目名称，项目介绍，以及作者等都可以默认，vue-router是官方提供的路由插件，在这里我选择Y。

接下来的几项我都选择N，这里大概是指是否安装规范代码格式检查，由于是测试，这里就不安装了。然后就静静的等待它安装完成：

安装完成

你也可以输入webpack -v检查安装正确与否

这是在看目录文件夹，项目结构已经出来了

接下来执行npm install命令，这一步也可以省略，不过为了后面项目的开发，建议安装。

接下来就是见证奇迹的时刻到了，输入命令npm run dev，在本地查看项目。

然后就在浏览器中输入http://localhost:8080，若显示如图所示，则代表你已经安装成功了。
